# VIBE

**V**LLM **I**nference **B**enchmarking **E**ngine

A reproducible benchmarking framework for evaluating modern LLM inference systems.

---

## Features

- 🚀 vLLM Benchmarking
- ⚡ Prefill-Decode Disaggregation (PD)
- 📊 Throughput / TTFT / ITL / Goodput Analysis
- 💾 KV Cache Evaluation
- 🐳 Docker Compose Deployment
- 🖥️ AMD MI300X (ROCm) Support
- 📈 Experiment Reproducibility
